# QG Games Developer Website

Static developer website for GitHub Pages.

## Deploy on GitHub Pages

1. Upload `index.html`, `style.css`, and `script.js` to your GitHub Pages repository.
2. In GitHub, go to **Settings → Pages**.
3. Set source to your main branch and root folder.
4. Save and open the published URL.

## Customize

- Replace the App Store URL in `index.html` if your exact Block Stack QG App Store link is different.
- Update the text, name, and project list as you release more games.
